== Description ==

Simple way to displaying Recent Post Widget with date, post thumbnail. Its highly customizable for every theme developer to make change as his / her need.

= Features =
* Specify widget title
* All new, better Recent Posts widget
* Control number of posts displayed
* Control number of posts to skip (offset)
* Optionally display post thumbnails
* Optionally display post date
* Customize thumbnail size
* Choose post type to display
* Multisite capable

== Installation ==
1. Upload the simple-recent-posts-widget folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Appearance > Widgets
4. Select "Simple Recent Posts" on sidebar.
